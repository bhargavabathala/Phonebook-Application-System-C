# 📒 Phonebook Application System

A console-based Phonebook Application System developed in C language 
as part of my Bachelor's degree project. This project demonstrates 
the use of file handling, structures, and basic CRUD operations in C.

---

## 🎯 Features

- ➕ Add new contact (Name, Phone Number, Address)
- 🔍 Search contact by name or number
- ✏️ Update existing contact details
- ❌ Delete a contact
- 📋 Display all contacts
- 💾 Data stored using File Handling

---

## 🛠️ Technologies Used

- Language   : C
- Concepts   : Structures, File Handling, Arrays, Functions
- Compiler   : GCC / Turbo C
- Platform   : Windows / Linux

---

## ⚙️ How to Run

### Using GCC Compiler:
```bash
gcc phonebook.c -o phonebook
./phonebook
```

### Using Online Compiler:
- Go to https://www.onlinegdb.com
- Paste the code and click Run
